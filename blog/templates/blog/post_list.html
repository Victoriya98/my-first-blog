Исследование возможностей языка Python в Web-разработке

Цель:

Исследовать и оценить возможности языка программирования Python в сфере Web-разработки.

Задачи:

1.Изучить инструменты и возможности языка программирования Python.
2.Исследовать Django - фреймворк для web-приложений на Python.
3.Самостоятельно создать web-сайт c подробным описанием процесса.
4.Оценить возможности языка для быстрой разработки небольших приложений, решающих поставленные задачи.
